Sequences and Sets
==================

Sequences (:doc:`lists` and :doc:`tuples`) and sets (:ref:`set` and
:ref:`frozenset`) essentially differ in the following properties:

+---------------+---------------+---------------+---------------+---------------+
| Data type     | :term:`mutable| ordered       | indexed       | duplicates    |
|               | <immutable>`  |               |               |               |
+===============+===============+===============+===============+===============+
| List          | ✅            | ✅            | ✅            | ✅            |
+---------------+---------------+---------------+---------------+---------------+
| Tuple         | ❌            | ✅            | ✅            | ✅            |
+---------------+---------------+---------------+---------------+---------------+
| Set           | ✅            | ❌            | ❌            | ❌            |
+---------------+---------------+---------------+---------------+---------------+
| Frozenset     | ❌            | ❌            | ❌            | ❌            |
+---------------+---------------+---------------+---------------+---------------+
