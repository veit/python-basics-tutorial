Boolean values
==============

Boolean values are used in the following examples:

.. code-block:: pycon

    >>> x = False
    >>> x
    False
    >>> not x
    True

.. code-block:: pycon

    >>> y = True * 2
    >>> y
    2

Apart from their representation as ``True`` and ``False``, Boolean values
behave like the numbers ``1`` (``True``) and ``0`` (``False``).
